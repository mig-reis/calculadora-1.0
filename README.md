# Calculadora em Python

Projeto simples em Python que solicita o nome do usuário e realiza a multiplicação de dois números inteiros.

## Tecnologias utilizadas
- Python
- Colorama

## Funcionalidades
- Entrada de dados pelo terminal
- Tratamento de erros (ValueError)
- Saída colorida no terminal

## Como executar
```bash
pip install colorama
python desafio1.py
